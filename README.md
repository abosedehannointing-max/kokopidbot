# Telegram Auto Post Bot - No Database Required

Automatically generates and posts content about any topic to Telegram channels. Everything runs in memory!

## Features

- ✅ **No database needed** - All data stored in RAM
- ✅ Automatic content generation (10+ templates)
- ✅ Posts every 90 minutes (~16 posts/day)
- ✅ Supports 1-30 day campaigns
- ✅ Interactive and quick setup
- ✅ Progress tracking
- ✅ Auto cleanup when campaign ends

## Quick Deploy on Render

1. Fork this repo to GitHub
2. Create bot on Telegram via @BotFather
3. Deploy on Render (Free tier works!)
4. Add environment variable: `TELEGRAM_BOT_TOKEN`
5. Done!
